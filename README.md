# sitewcurriculos

Landing page da **WCurrículos**, serviço que entrega currículos profissionais em PDF diretamente pelo WhatsApp.

## Sobre

Site estático em HTML/CSS/JavaScript com:

- Apresentação do serviço e como funciona
- Depoimentos de clientes
- Planos disponíveis
- Formulário de preenchimento dos dados do currículo

## Como usar

Abra o arquivo `index.html` em um navegador para visualizar o site localmente.
